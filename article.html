Телефонный справочник

Назад Скачать
Компонент предоставляет интерфейс поиска по сотрудникам по полям Имя, Фамилия, Отчество, E-mail, Должность, Мобильный телефон и по пользовательскому свойству UF_PHONE_ADD(Добавочный телефон), если оно есть в системе. Строка поиска + алфавит (поиск по первой букве).
Обязательно подключить Jquery в header.php или footer.php шаблона сайта.
Пагинация происходит по прокрутке вниз страницы (lazyload)

          1) Убедиться, что в папке /local/components/ нет компонентов с аналогичным названием и в корне сайта нет папок с совпадающими названиями.
  2) Не обязательно, но можно создать для пользователей пользовательское свойство UF_PHONE_ADD (добавочный телефон), если нужно искать по добавочному телефону.
